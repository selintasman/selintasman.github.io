# Don Quijote Project

This is a final project for SPA454 at the University of Toronto, featuring a blog inspired by Cervantes' famous novel Don Quijote. 

This website was based on the [Clean Blog](https://startbootstrap.com/theme/clean-blog/) template initially created by [Start Bootstrap](https://startbootstrap.com/).
